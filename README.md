## 📊 Dataset
- **Name:** Students Performance in Exams
- **Source:** Kaggle
- **Rows:** 1000
- **Columns:** 8 (gender, race/ethnicity, parental education, 
lunch, test prep course, math/reading/writing scores)

---

## 🔍 What I Did
- Loaded dataset using Pandas
- Checked and confirmed zero missing values
- Checked and confirmed zero duplicate rows
- Generated 3 summary statistics
- Created 3 visualizations (Histogram, Bar Chart, Box Plot)

---

## 📈 Key Findings
- Average Math Score: 66 | Reading: 69 | Writing: 68
- Females perform better in Reading & Writing
- Males perform better in Math
- Only 358 out of 1000 students completed test preparation course
- Students who completed test prep scored higher overall
